How to Send Email Using Telnet in Kali Linux

I think everyone already know how to send an email especially the conventional email, that is why I made this tutorial about how to send email using telnet in kali linux.

The way we sending email in this tutorial is a little different than sending email using Gmail or Yahoo, because we will try to sending email by using a command prompt or terminal.

Do not confused with the title How to send email using telnet in kali linux, because the telnet protocol was same for all operating system.

Before we start:

Make sure we have access to a SMTP server. Usually when you subscribe the internet connection to ISP, they will give you access to their SMTP server or they give you one email address for you to use (In Indonesia some internet provider usually will give you 1 email account when you subscribe, that's my experience. The good and also the bad thing is their SMTP(Simple Mail Transfer Protocol) server doesn't need an authentication if you are inside the ISP network.
